Sveriges är snart bäst i världen. DIGG driver på

Code based on Wikidata Card Game Generator available at GITHUB

Draft av hur en DIGG Dashboard skulle kunna se ut som bättre kommunicera status på Öppen Data

Tanken är att DIGG om några veckor har gjort Sverige bäst i världen på öppen data. Ryktet säger att Sveriges kommuner redan kanske denna vecka når målet.

Genom att klicka på ett kort så skulle man kunna:

  • starta Google Lighthouse - exempel test på denna sida
    • En analys av prestanda görs:
      • Critical Rendering Path (CRP)
      • Uppskatta Input Latency dvs. allt över 100ms anses segt
      • Speed index skapas och kan jämföras med alla andra arbetslivsmuseerse speedline
    • Andra kritiska saker kontrolleras som Wikipedia sida/ Google registrerad / Wikidata postens kvalitet
    • Stöd för FAIRDATA link
    • Skapa olika DIGG index
      • grad av kopplad till länkade data
      • material öppet tillgängligt cc-0
      • materialet finns hos andra aktörer
