Arbetslivsmuseer - Google Lighthouse



Hur det kommer att fungerar när Tore levererat? Från Wikidata hämtas Arbetslivsmuseer med egenskaper, och kort visas

Genom att klicka på ett kort så kommer:

  • Google Lighthuse att köras
     
  • En analys av prestanda görs
     
  • Critical Rendering Path (CRP)
     
  • Uppskatta Input Latency dvs. allt över 100ms anses segt
     
  • Speed index skapas och kan jämföras med alla andra arbetslivsmuseerse speedline
     
  • Andra kritiska saker kontrolleras som Wikipedia sida/ Google registrerad / Wikidata pos
