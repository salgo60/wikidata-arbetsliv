Sveriges digitaliserade kulturarv hur mår det?

Draft av hur en Digisam Dashboard hur dagsläget kan kommuniceras

Code based on Wikidata Card Game Generator

Tanken att är att Digisam kommunicerar på ett bättre/tydligare sätt så vi kan förstå och lära av varandra.

Nedan vad man kan på 1 timme få ihop med Wikidata och sedan kan exempelvis Google Lighthouse enkelt kopplas in



Genom att klicka på ett kort så skulle man kunna:

  • starta Google Lighthouse - exempel test på denna sida
    • En analys av prestanda görs:
      • Critical Rendering Path (CRP)
      • Uppskatta Input Latency dvs. allt över 100ms anses segt
      • Speed index skapas och kan jämföras med alla andra arbetslivsmuseerse speedline
    • Andra kritiska saker kontrolleras som Wikipedia sida/ Google registrerad / Wikidata postens kvalitet
    • Stöd för FAIRDATA link
    • Skapa olika Digisam index
      • grad av kopplad till det Svenska Kulturarvet
      • material öppet tillgängligt cc-0
      • materialet finns hos andra aktörer Digitalt museum / K-samsök....
      • aktiviteter med Maskininlärning/AI link
